# Folgende Rezepte wurden angepasst

* Alle Coral-Blöcke (auch  Tote) können aus ihren Jeweiligen "normalen" Korallenarten im 2x2 gecraftet werden
* Zum Craften des Anvils sind die äußeren zwei Blöcke durch normale Ingots ersetzt
* Blaueis kann man statt im 3x3 im 2x2 mit Packeis machen
* Packeis kann man statt im 3x3 im 2x2 mit Eis machen
* Eis kann man im 3x3 mit Schneebällen craften
* Sand kann man im 2x2 aus Gravel Craften
* Gravel kann man im 2x2 aus Cobblestone Craften
* Cobbled Deepslate kann man im Stonecutter aus Deepslate machen
* Cobblestone kann man im Stonecutter aus Stone machen
* Blackstone kann man im 2x2 aus Basalt machen
* Rezept für Crafter entfernt
* Rezept für Enderchest entfernt
